Best estimate calculation in a sensor network in the presence of redundancy
============================================================================

The main functions included in the file *redundancy1.py* are:

#. :mod:`calc_best_estimate`: Calculation of the best estimate for a given set of estimates with associated uncertainty matrix.
#. :mod:`calc_lcs`: Calculation of the largest subset of consistent estimates of a measurand.
#. :mod:`calc_lcss`: Calculation of the largest subset of sensor values that yield consistent estimates of a measurand linked to the sensor values by a linear system of equations.
